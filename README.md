# Bookmarker

A small front-end bookmark app built with plain HTML, CSS, and JavaScript.

## Overview

This project lets you save website names and URLs in the browser using `localStorage`. Saved bookmarks are shown in a list where you can visit or delete them.

## Features

- Add a website name and URL
- Save bookmarks in the browser with `localStorage`
- Prevent duplicate entries with the same website name
- Open saved links directly from the list
- Delete bookmarks from the list
- Submit with the button or by pressing `Enter`

## Project Structure

- `index.html` - page structure
- `style.css` - app styling and responsive layout
- `app.js` - bookmark logic and `localStorage` handling

## How to Run

No build step or installation is required.

1. Clone or download the project.
2. Open `index.html` in your browser.

You can also serve it locally with any simple static server if you prefer.

## Notes

- Bookmarks are stored only in the current browser.
- Data will not sync across devices or browsers.
